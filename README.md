## El problema que resuelve

La mayoría de herramientas de gestión de proyectos son demasiado complejas para el día a día de un negocio pequeño, o demasiado simples para sostener un proceso de mejora real.

KanbanPro cierra esa brecha. Es una plantilla de Notion lista para usar que integra tres metodologías probadas en una sola interfaz, pensada para usarse desde el celular.

---

## Lo que incluye

### Módulos principales

**Tablero Kanban dinámico**
Columnas adaptativas que escalan con la complejidad del proyecto. Sin límite de tareas. Sin curva de aprendizaje.

**Hoja de ruta DMAIC**
Plantillas paso a paso para cada fase de Six Sigma: Definir, Medir, Analizar, Mejorar y Controlar. Todo en un flujo guiado.

**Registro Kaizen**
Sistema de captura rápida para micro mejoras sobre la marcha. Ideal para equipos que operan en campo.

### Bonos incluidos

**Espacio de trabajo visual**
Panel con analítica drag and drop para visualizar el estado del proyecto en tiempo real.

**Plano de Notion completo**
Plantilla organizacional para sincronizar datos del móvil con el escritorio y mantener todo centralizado.

Este repositorio contiene la infraestructura completa del producto:

Pitch decks        Node.js, PPTXGenJS (presentaciones en PPTX)
Lead tracker       Python, OpenPyXL (seguimiento en Excel)
